# My Streamlit App

This is a simple Streamlit application.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-repo/streamlit_project.git
    ```
2. Navigate to the project directory:
    ```bash
    cd streamlit_project
    ```
3. Install the package:
    ```bash
    pip install .
    ```

## Usage

Run the app with:
```bash
run-app
